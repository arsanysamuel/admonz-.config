Including config files for:
- X11 (with startup programs)
- bash
- nnn
- Arduino CLI
- dunst (Notification Daemon)
- mimi (MIME filetypes)
- mpd
- mpv
- ncmpcpp
- neomutt
- newsboat
- neovim
- qutebrowser
- ranger
- transmission
- stig (transmission terminal client)
- sxhkd
- sxiv
- youtube-dl
- zathura
- pikaur (AUR helper)
